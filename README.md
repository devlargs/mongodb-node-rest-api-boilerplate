# RAMEN

## RAMEN stands for Rest-API-Mongo-Express-Node
 
### Installation

#### Clone the repository.
```
git clone git@github.com:devlargs/ramen.git
```

#### Edit the port on ./server.js (1234 is the port by default)

#### Install Dependencies (yarn or npm).
```
yarn install
```

#### Then 
```
yarn run dev
```
